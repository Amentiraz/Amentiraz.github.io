# 最喜欢的歌手 / 创作者

Guiano 很喜欢他清新的曲风和歌词中积极的意味
John Mayer 让我认识到吉他的魅力和蓝调 / R&B 魅力的欧美歌手，也是我唯一关注的欧美歌手
郑晟河 最喜欢的指弹吉他手，改变了我对演奏与音乐的认识，我的吉他上现在还存有他的签名
梶浦由记 在动画配乐中最喜欢的一位，个人认为虽然不如菅野よう子在日漫中影响力那么大，但是单论风格是我最喜欢的一档
