# 人生经历

2003 年生人，本科专业方向是计算机科学与技术，研究生研究领域大致在生物信息学领域。

人生经历也算丰富多彩，甚至可以算是跌宕起伏：

小学以最后一名进入，最后以第一名补录进入市内最好的初中。

初中以补录进入年级最差的班，最后好歹以名单倒数几名直升高中。

高中一开学以零基础三个月的 OI 生涯拿下 NOIP 省二，而后在高二以完全形态发挥失常，做不出来第一题，在 CSP 折戟沉沙。

沉沦了数个月，随波逐流，不愿学习，好在最后在朋友的帮助下决定再试一把，努力学习主课最终在一诊超越班级第二名 20 分夺得当次考试第一。

然而第一带来的不是荣誉，而是巨大的压力与同学的嫉妒，清醒的认识到备战高考的过程不是提升自己而不过是异化自我的过程罢了。

虽说在备战高考时经历了学习成绩的巨大落差、心脏不适、耳朵发炎等大大小小的疾病，好歹最后稳住心态，普普通通的发挥，普普通通的失误，进入国内某知名 211。

进入大学后并没有像他人那般努力学习主课，反倒 “不务正业”：

  • 参加了学校举办的散打比赛，在第一轮遇到本组准冠军，即便我多次把他逼出场外，甚至将其击倒，最终以点数之差失败。
  • 参加 ICPC 省赛，最终三人小队未能正确理解第一题题意，爆零失败。
  • 每天坚持练习吉他，但练习的是最不受待见的指弹，又难又不好表演，但好歹技术是上来了。目测再学个 10 年就能成为 Guitar Hero 了！
  • 学习哲学，先了解了西方哲学史，又自主了解康德、黑格尔等哲学家，最终发现现象学更适合我，开始系统学习现象学。
  • 参加了美国大学生数学建模竞赛拿到特等奖（F），逐渐取得了自信
  • 热爱动漫，大学看了快 400 部动漫，成为了冻鳗糕手。

没有琢磨什么专业和方向怎么博弈，最后进了最卷的一个专业当中最卷的一个方向。

本来都准备好考研了，然而造化弄人，由于前面的奖项还行，最终以专业第一成绩完结了大学的学习生活，保研至本校的研究生。

博客在 2022 和 2023 断更过两年，那两年我去思考人生去了，思考完人生感觉人生丰富多彩，值得记录，于是又返回来重新开始记录博客，便有了这个网站。

希望我的博客能为你带来帮助.

# 兴趣爱好

# 综合格斗

虽然现在不打比赛了，但仍然观看 UFC 的比赛。

最喜欢的选手是托尼・弗格森，从他那里我学习并实战运用了第一个柔术技术：三角绞，他比赛时不屈不饶的精神和独特自创的打法也使我很受用。然而从我开始看他开始便是他 12 连胜的终结和 7 连败的开始，令人感慨。

# 动漫

bangumi

动漫记录平台

我看动漫大致分为下面几个阶段

  • 100 部时，看了几部大热动漫便认为自己看的已经足够多了，然而既没有影视观赏能力，对动漫也没有足够深入的了解
  • 200 部时，认识到动漫实在太多，开始系统补知名的动漫，这个阶段开始看别人的解读，逐渐培养起自己对动漫的审美
  • 300 部时，又膨胀了，网上出名的作品几乎看了个遍，没看过的也足够了解相关必要的知识，尝试琢磨分镜、剧本等较为专业的观影知识，但越是这样却发现自己越难以把握动画
  • 400 部时，看了真正的名作，有了自己的审美，不再追求动画的制作水平是否高超，而是能给自己带来的情感体验，已经能做出能类似于网络上影评的独创的分析了
  • 500 部时，知道自己的审美喜好，对于不是自己喜好的作品也能客观分析，回归最开始看动画的纯粹的观看体验
  • to be continue

目测看到大概 700 部左右的时候，我的审美，评价作品的手法会真正意义上的成熟。

对我影响最大的几部作品：

# 三月的狮子

bangumi

三月的狮子

高三每周一集，陪伴我度过了最折磨的一段时光，也让我认识到情感的重要性

# 物语系列

bangumi

物语系列

我完整看完的第一个长篇，也是我看日本动漫的入门之作

# FLCL

bangumi

FLCL

这部动画富有激情，在我高中和大学时每次观看它都有着截然不同的观影体验，每当我陷入低谷时我都会来看看这部动画，汲取能量

# EVA

bangumi

EVA：真心为你

以我现在的视角，这部动画有点显得肤浅了，但它在爱与存在这两个名词的论述上给予了我极大的启发，也是我在哲学上走向现象学的原因之一

以后的我可能没有那么多时间看动漫了，但希望自己仍然保留对它的热爱吧。
